# Enforcer LLM

A demonstration of using an LLM to generate enforcers for [Delegatable](https://github.com/delegatable/delegatable-sol).

## Setup

Install [llm by Simon Willison](https://github.com/simonw/llm)

## Usage

Run `./start.sh $YOUR_ENFORCER_DESCRIPTION`

Example:

`./start.sh "Only allow during odd blocks"
